════════════════════════════════════════════════════════════════════════════════
                        REGLAS DE ORO
════════════════════════════════════════════════════════════════════════════════

1. NO SALTES PASOS
   Si quieres Dashboard (Paso 3) antes de terminar Auth (Paso 2),
   el proyecto no funcionará. Sigue el orden.

2. LEER COMPLETO ANTES DE EMPEZAR
   No hagas "leo línea 1, empiezo a codear". Lee el documento completo.

3. COMMIT A GIT DESPUÉS DE CADA PASO
   Si algo se rompe, puedes volver atrás.

4. PREGUNTAS ESPECÍFICAS > PREGUNTAS GENÉRICAS
   "No funciona" ❌
   "npm run dev da error X en línea Y" ✅

5. ACTUALIZA ESTA DOCUMENTACIÓN
   Si encuentras errores o mejoras, dilo.

6. NO SALTES A MONGODB O POSTGRES
   Firebase es suficiente. Si después quieres cambiar, lo hacemos.

7. VERIFICA CADA PASO LOCALMENTE
   npm run dev debe funcionar después de cada paso.

# SoftPlugin Pro — Sitio Web

Sitio web profesional para presentar y distribuir SoftPlugin Pro.

## Estructura del Proyecto

```
├── index.html              # Página principal
├── css/
│   └── styles.css          # Estilos personalizados
├── js/
│   └── main.js             # JavaScript funcional
├── assets/
│   ├── img/                # Imágenes e iconos
│   │   └── favicon.svg     # Favicon del sitio
│   └── downloads/          # Archivos de descarga
│       └── Setup.exe       # Instalador (colocar aquí)
└── README.md               # Este archivo
```

## Cómo Usar

1. **Abrir localmente**: Abre `index.html` en tu navegador.
2. **Subir a hosting**: Sube toda la carpeta a tu servidor web (Netlify, Vercel, hosting tradicional, etc.).

## Archivo de Descarga

Coloca el archivo `Setup.exe` dentro de la carpeta `assets/downloads/`. El botón de descarga apunta a esa ruta automáticamente.

Para actualizar el instalador, simplemente reemplaza el archivo `Setup.exe` con la nueva versión.

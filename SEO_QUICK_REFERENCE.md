## ❌ PROBLEMAS COMUNES & SOLUCIONES

### Error: "Dirección de sitemap no válida"
```
Causa: Archivo no en raíz o servidor no lo sirve
Verificar: curl -I https://fis-matt.com/sitemap.xml
Debe retornar: HTTP 200 + application/xml
```

### Error: "ERR_TOO_MANY_REDIRECTS"
```
Causa: Redirect infinito HTTP → HTTPS
Solución: GitHub Pages maneja esto automáticamente
```

### URLs no indexadas
```
Causa: robots NOINDEX tag presente
Verificar: Meta tags en index.html
Debe tener: <meta name="robots" content="index, follow">
```
